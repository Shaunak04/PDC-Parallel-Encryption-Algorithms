To run the code:

In Windows:

-Parallel implementation: 
  >Open VSCode terminal in the /Des/ directory
  >g++ -o p5 -fopenmp parallelimplementation.cpp
  >./p5
  
-Serial implementation: 
  >Open VSCode terminal in the /Des/ directory
  >g++ -o s5 -fopenmp serialimplementation.cpp
  >./s5
